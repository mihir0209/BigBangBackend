Vidtract Server

Server is running. This backend processes video content for the Vidtract browser extension.

Caching System

Results are now cached by video URL. When the same video is requested again, the cached results will be returned without reprocessing.

API Endpoints

POST /process

Process a video from a page URL and generate summary and transcript.

Request Body:

{
  "pageUrl": "https://www.youtube.com/watch?v=example",
  "pageTitle": "Example Video"
}

Response:

{
  "videoTitle": "Example Video",
  "videoSource": "YouTube",
  "summary": "Markdown formatted summary",
  "transcript": "Full transcript text"
}

Setup Instructions

  1. Install required dependencies: pip install -r requirements.txt
  2. Install Tesseract OCR for text extraction
  3. Set your Groq API key in app.py
  4. Run the server: python app.py

Browser Extension

The browser extension now uses a sidepanel interface and automatically processes videos when you navigate to a page.
